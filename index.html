# 内部推荐

# 公司介绍

西安卓派信息科技有限公司是 NEXT Trucking 在中国设立的子公司。NEXT Trucking (https://www.nexttrucking.com) 是一家总部位于美国加州洛杉矶的科技公司，公司专注于以卡车司机为主导的创新在线交易平台。

在这里你会得到：

  • 技术氛围浓厚的工作环境
  • 英文的办公环境，赴美国出差、进修、工作（我们会提供付费的英文课程给期望提升英文水平的小伙伴哦~~）
  • 舒适的办公环境：电动升降桌，人体工学座椅，4K显示器，MacBook（健康工作，职业病快走开~~）Switch，街机
  • 无限量免费水果、零食、饮品、咖啡（每天都是元气满满的一天~~）
  • Team building（春天踏青，夏天烧烤，秋天旅行，冬天温泉都是我们美好的记忆，还有不定期的运动趴，桌游趴，K歌趴，让你工作之余有更多丰富的活动体验~~）
  • 节日礼品（除了法定节日的礼品外，还会在你生活中每一个重要的日子里送出我们的祝福~~）
  • 节日假期（青年节，儿童节，妇女节，我们的假期一个都不会少哦~~）

# 在招岗位

# 中级前端工程师（7k-10k·13薪）

【岗位要求】

  1. 两年及以上Javascript领域的研发经验
  2. 熟练掌握前端基础技能，能够灵活运用HTML/CSS/JS相关知识
  3. 熟练掌握前端相关调试技术，如Chrome Dev Tools、DHC等
  4. 能够使用 React / React Native框架构建前端应用
  5. 有良好的代码编程习惯、遵守一定编程规范并有良好的团队沟通和合作能力
  6. 善于思考，能独立分析和解决问题，对技术有热情及好奇心，关注技术发展方向，能够自我驱动
  7. 良好的英语口语和书面表达能力

【加分项】

  1. 流利的英语口语
  2. 熟悉模块化、前端编译及构建工具的配置与使用
  3. 了解服务端相关技术（如NodeJS/Java或其他语言），对数据结构和算法有一定了解
  4. 有自己的技术博客并定时更新
  5. 开源爱好者/贡献者、Github/Stackoverflow等技术社区活跃者、Facebook/Twitter等互联网产品深度用户

# 高级前端工程师（15k-20k·13薪）

【岗位要求】

  1. 三年及以上Javascript领域的研发经验
  2. 高效使用Webpack，Grunt，Gulp等Web开发工具
  3. 能够使用 React / React Native 框架构建前端应用
  4. 能够对前端项目进行长期的开发迭代及维护，并了解其运行机制
  5. 能够构建可重用组件并实现组件化生产
  6. 能够根据前端项目制定规则并输出英文文档
  7. 了解一门服务器端相关技术，并有一定的实战经验
  8. 良好的英语口语和书面表达能力

【加分项】

  1. 流利的英语口语
  2. 有指导团队成员的相关经验
  3. 了解移动端开发相关技术
  4. 有自己的技术博客并定时更新
  5. 开源爱好者/贡献者、Github/Stackoverflow等技术社区活跃者、Facebook/Twitter等互联网产品深度用户

# React Native工程师（14k-17k·13薪）

【岗位要求】

  1. 两年以上的React Native开发经验
  2. 精通React Native/ Redux
  3. 掌握RN 和 Native 之间的通信
  4. 有较强的逻辑思维能力和沟通能力

【加分项】

  1. 熟悉 iOS开发
  2. 熟悉 Java 语言
  3. 有自己的技术博客或网站
  4. 参与开源项目

# iOS高级开发工程师（15k-20k·13薪）

【岗位要求】

  1. 三年以上的iOS开发经验
  2. 掌握iOS发版流程

  4. 有较强的沟通能力，更高效率的和其他团队合作
  5. 可使用英语直接沟通

【加分项】

  1. 熟悉 React/React Native 语言
  2. 熟悉 Java/Android 语言
  3. 有自己的技术博客或网站
  4. 参与开源项目

【加分项】

  1. 互联网研发经验
  2. AWS, 阿里云等云服务平台使用经验
  3. 有自己的技术博客并定时更新
  4. 技术涉猎广，有以下产品使用或研发经验：Kafka, ELK, kubernetes等
  5. 各种MOOC认证

  8. 责任心强，能独立分析和解决问题，具备良好的团队合作精神和承压能力

【岗位要求】：

  1. 两年以上互联网应用自动化测试开发经验
  2. 良好的编程能力， 熟练掌握一种开发语言以及一门脚本语言（Java,python,groovy等）
  3. 掌握常用web及移动端测试框架的应用及扩展(Selenium WebDriver, Cypress, Appium等)
  4. 熟悉CI/CD (CircleCI)
  5. 有较强的需求理解能力及业务逻辑分析能力
  6. 较强的英文交流沟通

【加分项】：

  1. 具备性能测试／压力测试经验

  4. 很强的编程能力，熟悉java,python等编程语言,掌握JavaScript，熟悉Linux系统和shell脚本语言，对CI/CD有开展应用经验
