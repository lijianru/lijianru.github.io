# 公司介绍

NEXT Trucking (https://www.nexttrucking.com)是一家总部位于美国加州洛杉矶的科技公司。公司致力于为托运人和卡车司机提供世界级服务，主营以卡车司机为主导的新型在线交易平台，深耕美国传统物流行业。其使命是解决万亿美元的难题，提供无痛货运。NEXT 的产品面向第一个以卡车司机为中心的市场并提供技术支持与革新，旨在让已经运送超过 50B 美元的货物一而这才只是刚刚开始。NEXT 于 2017 年 3 月获得信中利资本集团、 SheaVentures 、Simon Nixon 投资公司的 500 万美元的 A 轮融资，2018 年 1 月获得红杉资本的 2100 万美元的 B 轮融资，2019 年获得世界上最大的基础设施投资者之一的 Brookfeld Ventures 和红杉资本的 9700 万美元的 C 轮融资。如果您是在快节奏的技术环境中茁壮成长的充满激情的企业家类型，欢迎加入我们与来自 Airbnb 、 Google 、 Uber 和亚马逊的同事一起共事，我们共同将货运和物流提升到一个新的水平！西安卓派信息科技有限公司是 NEXT Trucking 在中国设立的子公司，为整个 NEXT Trucking 所运营的网站以及多种产品提供专业的研发、培训和技术支持。

在这里你会得到：

  • 技术氛围浓厚的工作环境
  • 英文的办公环境，赴美国出差、进修、工作（我们会提供付费的英文课程给期望提升英文水平的小伙伴哦~~）
  • 舒适的办公环境：电动升降桌，人体工学座椅，4K 显示器，MacBook（健康工作，职业病快走开~~）Switch，街机
  • 无限量免费水果、零食、饮品、咖啡（每天都是元气满满的一天~~）
  • Team building（春天踏青，夏天烧烤，秋天旅行，冬天温泉都是我们美好的记忆，还有不定期的运动趴，桌游趴，K 歌趴，让你工作之余有更多丰富的活动体验~~）
  • 节日礼品（除了法定节日的礼品外，还会在你生活中每一个重要的日子里送出我们的祝福~~）
  • 节日假期（青年节，儿童节，妇女节，我们的假期一个都不会少哦~~）

# 在招岗位

# 前端开发工程师

【岗位职责】

  • 参与需求沟通分析及技术方案设计
  • 根据技术方案完成编码及单元测试
  • 维护和升级现有程序，并快速定位，重构和修复现有软件缺陷
  • 运用 Javascript 与后台协作实现对前台网站系统的开发及运维

【岗位要求】

  • 2 年及以上 Javascript 领域的研发经验
  • 熟练掌握前端基础技能，能够灵活运用 HTML / CSS / JS 相关知识
  • 熟练掌握前端相关调试技术，如 Chrome Dev Tools 、 DHC 等
  • 能够使用 React 框架构建前端应用

# 高级前端开发工程师

【岗位职责】

  • 使用 React 实现平台功能的开发
  • 使用 Redux 或其他主流库进行数据协调、交互及状态管理，并对其生命周期具备一定见解
  • 使用 Webpack 对项目进行合理优化及辅助
  • 充分了解 React 组件的生命周期， React Hooks 实现机制，开发并使用可重用的 React 组件

【岗位要求】

  • 3-5 年及以上 Javascript 领域的研发经验
  • 高效使用 Webpack , Grunt , Gulp 等 Web 开发工具
  • 能够使用 React 框架构建前端应用
  • 能够对前端项目进行长期的开发迭代及维护，并了解其运行机制

# 自动化测试工程师

【岗位职责】

  • 负责 Web 端或移动端产品的业务项目测试，完成自动化测试脚本的编写
  • 熟悉主流 Web / Mobile 测试框架- 熟悉 Jira ,敏捷研发流程

【岗位要求】

  • 有 web 端以及移动端应用的端对端自动化测试经验
  • 熟悉 node . js / Mocha 等或具有研发项目经验优先，至少擅长一门编程语言
  • 移动端自动化测试或底层测试优先，有 appium 等相关框架项目经 验

# 高级测试开发工程师

【岗位职责】

  • 开发以及优化和维护现有测试框架
  • 自动化质量平台的维护和优化
  • 测试工具类的研发，提升测试效率
  • 协助建设测试体系，完善测试流程

【岗位要求】

  • 较好的编程能力，掌握一种或多种编程语言
  • 熟悉 Java , node . js , appium , mocha 等优先 3.熟悉移动端应用的自动化测试
  • 有丰富的 CI / CD 经验
