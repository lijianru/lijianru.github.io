# 高阶组件（HOC）

高阶组件（HOC）是react中用于复用组件逻辑的一种高级技巧。HOC自身不是React API的一部分，它是一种基于React的组合特性而设计成的设计模式。具体而言。高阶组件是参数委组件，返回值为新组件的函数。

  • HOC不会修改传入的组件，也不会使用继承来复制其行为
  • HOC将组件包装在容器组件中来组成新组件，HOC是纯函数，没有副作用
